## 傍晚：完成 AI 双模式架构（两手准备）

用户确认 MuseRock 是**本地优先**的创作工具，未来可能支持云端。据此实现了：

### 新建文件
- `apps/web/src/services/ai-provider.ts` — AI 调用抽象层，支持 Local（直连 API）和 Cloud（后端代理）两种模式
  - `LocalAIService`：本地模式，API Key 存 localStorage，直接调 Gemini/OpenAI/Anthropic
  - `CloudAIService`：云端模式，调后端 `/ai/*` 代理端点
  - `createAIService()` / `createApiKeyService()` 工厂函数根据模式返回正确实例
  - `getAIMode()` / `setAIMode()` 模式切换（持久化到 localStorage）

### 修改文件
- `apps/web/src/App.tsx`：
  - 引入 `ai-provider.ts` 抽象层
  - 初始化 `AIService` / `ApiKeyService` 改用工厂函数
  - 设置面板新增 "AI Connection Mode" 切换按钮（Cloud/Local）
  - API Key 存储提示文本根据模式动态显示
- `apps/api/src/ai/adapters/adapter.factory.ts`：注册 Anthropic 适配器
- `docs/NEXT_ACTIONS.md`：英文更新，标记 P0/P1 完成项，新增架构图

### 已有代码确认可用（无需改动）

- `apps/api/src/api-keys/api-keys.service.ts` — AES-256-GCM 加密存储已实现
